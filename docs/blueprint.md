# **App Name**: Udakargalam

## Core Features:

- Homepage Hero: Display a looping ocean video with a fallback CSS animation if the video fails or the user prefers reduced motion.
- i18n Support: Implement language switching between English and Sanskrit using a locale toggle, persisting the user's choice.
- Grantha PDF Display: Show a grid of language cards that, when clicked, open the corresponding PDF in a new tab. The thumbnail shown on the card is lazy loaded.
- SEO Optimization: Include meta tags, preconnect, and other features for maximum SEO.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to evoke the ocean depths and create a sense of seriousness and depth.
- Background color: Very light blue (#F0F4F8) to provide a clean, academic backdrop.  This background should promote readability.
- Accent color: Muted gold (#A9927D) to represent knowledge and tradition, and contrast subtly with the blues and provide visual interest to key elements.
- Headline font: 'Literata' serif with good Devanagari support, lending a classical feel, and suitable for headlines
- Body font: 'Inter' sans-serif font provides a modern feel for longer text.
- Use Heroicons for minimalist glyphs in the header to represent different pages.
- Masonry layout for Grantha page card grid. Cards slightly lift on hover.
- Subtle fade animation when switching between languages.
